# colorless-image-view
Get a colored image from the user and show the grayscale version side by side.

## Problem broken into smaller parts:

1. Take an image as input from user.  
    - A simple file input tag will do.
    - Since, all is happening in local memory, no need of a form.

2. Make a local copy and convert to black and white.
    -

3. Show the edited image side by side to the original.
    - Both of the input and output images should be in it's own canvas.
    - I don't know why, I just want to.
    - Actually, the output must be in canvas. So I thought, "Why not the input too?"
